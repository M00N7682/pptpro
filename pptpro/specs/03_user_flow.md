# 유저 플로우 - 장표 제작 기능

User → (입력 화면) → 주제/타겟/목표 입력
 → LLM과 대화하며 스토리 확정
 → 목차/헤드메시지 리스트 표시
 → 각 장표별 템플릿 선택 UI
 → LLM이 슬라이드 내용 초안 생성
 → 사용자 수정 / USER_NEEDED 위치 표시
 → PPT 파일 생성/다운로드

# 유저 플로우 - 장표 보완 기능

User → 기존 PPT 업로드 or 이미지 붙여넣기
 → OCR → LLM 구조 분석
 → 수정/추가되어야 할 컴포넌트 UI로 표시
 → 사용자 선택
 → LLM 내용 생성 삽입
 → PPT 파일 업데이트 다운로드
