# 서비스명
PPT Pro

# 서비스 개요
컨설턴트와 기획자가 PPT 장표를 효율적으로 제작하고 수정할 수 있도록 돕는 AI Agent 서비스.

# 핵심 문제 정의
- PPT 작성은 많은 시간이 듦
- 템플릿/구조/스토리 설계에 인지적 리소스 필요
- 기존 AI 슬라이드 생성기는 '결과물 품질'이 낮으며, 전체의 맥락을 반영하지 못함

# 제공 가치 (Value Proposition)
- LLM과 단계화, 구조화된 PPt 설계를 통해 품질 높은 장표 생성
- 스토리라인 기반 장표 설계
- 상황/타겟/목표에 맞는 헤드메시지 자동 생성
- Case-box / As-is-To-be 등 **컨설팅 장표 템플릿 자동 매핑**
- 기존 장표에 대한 구조적 보완 자동화

# 핵심 기능 요약
1) 새로운 장표 제작 기능
2) 기존 장표 보완 기능

# 주요 사용자
- 컨설턴트
- 전략기획팀
- 공모전 & 발표자
