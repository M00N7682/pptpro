# 서비스명
PPT Pro

# 서비스 개요
PPT Pro는 컨설턴트와 기획자가 PPT를 작성할 때, 스토리라인 설계부터 장표 완성까지 전 과정을 AI Agent가 함께 제작·보완해주는 서비스임.  
사용자는 웹 화면에서 장표가 채워지는 과정을 실시간으로 보며, AI가 제시하는 슬라이드 구성·템플릿·인사이트를 선택하거나 수정할 수 있음.

# 문제 정의
- PPT 작성은 많은 시간과 인지적 리소스를 필요로 함.
- 스토리 구성과 템플릿 매핑은 전문성이 요구됨.
- 기존 AI PPT 생성 서비스는 구조적 논리·컨설팅형 장표 품질이 부족함.
- 사용자가 직접 수정해야 할 요소와 AI가 채울 요소가 섞여 있어 관리 어려움.

# 제공 가치
- 주제/타겟/목표 기반 스토리라인 자동 설계
- 각 장표의 헤드메시지 자동 생성
- 컨설팅 실무 기반 템플릿 추천 엔진
- USER_NEEDED / AI_GENERATED 자동 분리
- python-pptx 기반 실제 PPT 파일 자동 생성
- 웹 화면에서 실시간 장표 미리보기 및 수정

# 주요 사용자
- 컨설턴트  
- 전략기획팀  
- 기업 내부 발표 담당자  
- 공모전·피치덱 작성자  

# 핵심 기능 요약
1) 장표 제작 기능  
2) 장표 보완 기능  
3) 실시간 웹 기반 장표 미리보기  
4) PPT 파일 생성 및 다운로드
