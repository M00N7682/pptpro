# Glossary (PPT Pro 전용 용어 정의)

---

# Storyline
전체 발표의 논리적 흐름. Problem → Analysis → Insight → Solution → Impact 등.

# Head Message
각 장표의 핵심 주장. 장표 전체를 대표하는 1문장.

# Slide Template
장표의 레이아웃 유형.
- message_only
- asis_tobe
- case_box
- node_map
- step_flow
- chart_insight

# USER_NEEDED
사용자가 직접 조사·입력해야 하는 요소.
예: 시장규모, 경쟁사 수치, 통계자료, 내부데이터.

# AI_GENERATED
LLM이 자동으로 생성할 수 있는 요소.
예: 인사이트, 논리 연결문, 요약문, 구조화된 설명.

# Slide Components (슬라이드 요소)
AI가 채울 수 있는 세부 구성요소.
- Title
- Sub-Message
- Bullet Points
- Evidence Block
- Diagram Component
- Insight Box
- Action Guide
- Caption

# Rendering Engine
python-pptx 기반의 PPT 생성 엔진.

# Preview UI
웹 화면에서 슬라이드 요소가 실시간으로 채워지는 과정을 보여주는 인터페이스.
